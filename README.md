𝗗𝗶𝘀𝗲𝗮𝘀𝗲 𝗣𝗿𝗲𝗱𝗶𝗰𝘁𝗶𝗼𝗻 𝗨𝘀𝗶𝗻𝗴 𝗠𝗮𝗰𝗵𝗶𝗻𝗲 𝗟𝗲𝗮𝗿𝗻𝗶𝗻𝗴 (𝗠𝗦𝗰𝗜𝗧 𝗙𝗶𝗻𝗮𝗹 𝗬𝗲𝗮𝗿 𝗣𝗿𝗼𝗷𝗲𝗰𝘁)
🚀 𝗣𝗿𝗼𝗷𝗲𝗰𝘁 𝗢𝘃𝗲𝗿𝘃𝗶𝗲𝘄
This project is an AI-driven diagnostic support system designed to predict potential diseases based on patient-reported symptoms. By leveraging Ensemble Learning, the system provides a highly accurate (97%) prediction model to assist healthcare professionals in early-stage diagnosis.

🧠 𝗧𝗲𝗰𝗵𝗻𝗶𝗰𝗮𝗹 𝗜𝗺𝗽𝗹𝗲𝗺𝗲𝗻𝘁𝗮𝘁𝗶𝗼𝗻
The system utilizes a combination of three robust classification algorithms to ensure reliable results:
  • 𝗦𝘂𝗽𝗽𝗼𝗿𝘁 𝗩𝗲𝗰𝘁𝗼𝗿 𝗠𝗮𝗰𝗵𝗶𝗻𝗲 (𝗦𝗩𝗠): For high-dimensional data classification.
  • 𝗚𝗮𝘂𝘀𝘀𝗶𝗮𝗻 𝗡𝗮𝗶𝘃𝗲 𝗕𝗮𝘆𝗲𝘀: For probabilistic prediction.
  • 𝗥𝗮𝗻𝗱𝗼𝗺 𝗙𝗼𝗿𝗲𝘀𝘁: For ensemble-based decision making.
  • 𝗘𝗻𝘀𝗲𝗺𝗯𝗹𝗲 𝗟𝗼𝗴𝗶𝗰: A Majority Voting system was implemented to combine the outputs of the three models, significantly reducing error margins.

📊 𝗣𝗲𝗿𝗳𝗼𝗿𝗺𝗮𝗻𝗰𝗲 & 𝗥𝗲𝘀𝘂𝗹𝘁𝘀
  • 𝗔𝗰𝗰𝘂𝗿𝗮𝗰𝘆: Achieved a peak performance of 97%.
  • 𝗗𝗮𝘁𝗮𝘀𝗲𝘁: Analyzed 132 unique symptoms across various disease categories.
  • 𝗩𝗶𝘀𝘂𝗮𝗹𝗶𝘇𝗮𝘁𝗶𝗼𝗻: Implemented Confusion Matrices and Heatmaps using Seaborn and Matplotlib to validate model reliability.

  • 𝗘𝗻𝘃𝗶𝗿𝗼𝗻𝗺𝗲𝗻𝘁: Jupyter Notebook / Google Colab

In a professional context, this model serves as a prototype for:
  • 𝗧𝗲𝗹𝗲𝗺𝗲𝗱𝗶𝗰𝗶𝗻𝗲 𝗜𝗻𝘁𝗲𝗴𝗿𝗮𝘁𝗶𝗼𝗻: Automating initial symptom screenings.
  • 𝗥𝗲𝘀𝗼𝘂𝗿𝗰𝗲 𝗔𝗹𝗹𝗼𝗰𝗮𝘁𝗶𝗼𝗻: Helping clinics prioritize urgent cases based on predicted severity.
  • 𝗗𝗶𝗮𝗴𝗻𝗼𝘀𝘁𝗶𝗰 𝗘𝗳𝗳𝗶𝗰𝗶𝗲𝗻𝗰𝘆: Reducing the manual time required for preliminary symptom analysis.
