Disease Prediction Using Machine Learning (MScIT Final Year Project)
🚀 Project Overview
This project is an AI-driven diagnostic support system designed to predict potential diseases based on patient-reported symptoms. By leveraging Ensemble Learning, the system provides a highly accurate (97%) prediction model to assist healthcare professionals in early-stage diagnosis.

🧠 Technical Implementation
The system utilizes a combination of three robust classification algorithms to ensure reliable results:

Support Vector Machine (SVM): For high-dimensional data classification.

Gaussian Naive Bayes: For probabilistic prediction.

Random Forest: For ensemble-based decision making.

Ensemble Logic: A Majority Voting system was implemented to combine the outputs of the three models, significantly reducing error margins.

📊 Performance & Results
Accuracy: Achieved a peak performance of 97%.

Dataset: Analyzed 132 unique symptoms across various disease categories.

Visualization: Implemented Confusion Matrices and Heatmaps using Seaborn and Matplotlib to validate model reliability.

🛠️ Tech Stack
Language: Python

Libraries: Scikit-learn, Pandas, NumPy, Seaborn, Matplotlib

Environment: Jupyter Notebook / Google Colab

💼 Business & Healthcare Impact
In a professional context, this model serves as a prototype for:

Telemedicine Integration: Automating initial symptom screenings.

Resource Allocation: Helping clinics prioritize urgent cases based on predicted severity.

Diagnostic Efficiency: Reducing the manual time required for preliminary symptom analysis.
